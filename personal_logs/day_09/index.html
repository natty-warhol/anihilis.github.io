Switched My OS!!

Huzzah! I have done it! I am free from Solus Linux!

Don’t get me wrong, I love the damned Operating System, I just started feeling claustrophobic with the systemd and the perfect UX and clean UI… it was just…. too perfect…

So what did I switch to? I did a base install of Artix Linux! I love it so far! I went with the runit instance, because it seemed to have the best server/desktop useability. The hardest part was getting my Plex server set up, but I just had to learn how to set up a Docker service, and then I was off to the races!

On the laptop now, I have glorious HerbstluftWM! It’s my most favorite, whenever I got for a minimal install, I always use this WM, it hase the most sane controls I feel like, especially with the super easy to set up BASH config.

Now I just got to decide what else I should be doing with my server…. Anyone have any ideas?
