Switching to Artix Linux Personal Walkthrough

What’s my Mission?

I want to improve performance, security, and declutter my system by subverting Systemd. I also intend to encrypt my hard drive, as well as partial encryption of my internet traffic, whilst also using flatpak on large-library projects to avoid any system breakages.

USB Installation

Grab the ISO from ArtixLinux.org, install with:

sudo dd bs=4M if=path/to/image-runit.iso of=/dev/sdx status=progress oflag=sync

Installation – Prepare

When booting into a UEFI Laptop (any laptop made in the past 8 years just about), make sure it is in UEFI mode, despite how you might feel, it is a form of check, and good for software security. Without booting into the USB without UEFI mode, installation may not work.

Ensure that there is an internet connection. The easiest way to do so, is to use an ethernet cable. Otherwise…. shrugs I’m not sure what to do with this one, I didn’t really look into it because I’m lazy and they didn’t use netctl like Mainline Arch does.

Installation

Formatting

Got to use Luks Encryption!

parted /dev/sdx
mklabel gpt
mkpart ESP fat32 1Mib 512Mib
set 1 boot on
name 1 efi

mkpart primary linux-swap 512Mib 8Gb
name 2 swap

mkpart primary 8Gb 100%
set 3 lvm on
name 3 lvm
print

modprobe dm-crypt modprobe dm-mod

cryptsetup luksFormat -v -s 512 -h sha512 /dev/sdx3

cryptsetup open is the device to create under /dev/mapper
is the encrypted device

Example: cryptsetup open /dev/sdx3 luks_lvm

Partition Configuration

Create physical volume
pvcreate /dev/mapper/luks_lvm

vgcreate artix /dev/mapper/luks_lvm

lvcreate -n root -L XXG arch
lvcreate -n home -L XXG arch

Format partitions
mkfs.fat -F32 /dev/sdx1 mkfs.ext4 -L root /dev/mapper/arch-root mkfs.ext4 -L home /dev/mapper/arch-home mkswap /dev/sdx2

Mount Partitions

swapon /dev/sdx2
mount /dev/mapper/arch-root /mnt
mkdir -p /mnt/{home,boot}
mount /dev/sdx1 /mnt/boot
mount /dev/mapper/arch-home /mnt/home
mkdir /mnt/boot/efi

lsblk -f

Install Linux

basestrap /mnt base base-devel runit elogind-runit vim htop openssh linux-zen linux-firmware
fstabgen -U /mnt » /mnt/etc/fstab

artools-chroot /mnt

Base Configuration

ln -sf /usr/share/zoneinfo/America/Los_Angeles
hwclock –systohc

vim /etc/locale.gen
locale-gen

vim /etc/locale.conf

export LANG="en_US.UTF-8”
export LC_COLLATE="C”

Boot Loader

pacman -S grub os-prober efibootmgr
grub-install –target=x86_64-efi –efi-directory=/boot –bootloader-id=grub
grub-mkconfig -o /boot/grub/grub.cfg

Users

passwd

useradd -m user
passwd user

Network Configuration

vim /etc/hostname
insert hostname here
pacman -S dhcpcd networkmanager networkmanager-openvpn

All Done!

exit
umount -R /mnt
reboot
